    QA Questions

  1. Qus: What are the three components of Web?

    2. Ans: HTML, CSS and Javascript

  2. Qus: Define version control and give an example.

    Ans: Version control stores copies of the same document so that you can track all the changes being made to a document. Each new version may be a little different than the last. For example, version control can be helpful when using policies heavily. If an organization gets a question about past policies or when certain policies came into place they can search through all the versions of the policy that they had and use that to answer questions.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag used for the title of a webpage is <title>. For example, <title> LIB2024 QA Answers </title>.

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tag used to create a paragraph is <p>. For example <p>I like peanutbutter and jelly sandwiches </p>.

  5. Qus: Identify the odd one from these tags <span>, <p>, <a>,<i> and state the reason?

    Ans: The <i> is the odd one out because the other tags work to format the text and act on it structurally while the <i> acts on the look of the text.

  6. Qus: What are the tags required to create a navigation list in HTML? Give one example with two navigation lists.

    Ans: The tags required to create a navigation list are <nav> <a href> and <li> Examples include: Google Link and Amazon Link

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The tag used to create images is <img> and the attributes src, alt, width and height.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: The odd tag out is the <b> because it is used to alter text by making it bold while the other three tags are used to format a table.
