  2. Qus: Define version control and give an example.

    Ans: Version control stores copies of the same document so that you can track all the changes being made to a document. Each new version may be a little different than the last. For example, version control can be helpful when using policies heavily. If an organization gets a question about past policies or when certain policies came into place they can search through all the versions of the policy that they had and use that to answer questions.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag used for the title of a webpage is <title>. For example, <title> LIB2024 QA Answers <title>.

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tag used to create a paragraph is <p>. For example <p>I like peanutbutter and jelly sandwiches <p>.
